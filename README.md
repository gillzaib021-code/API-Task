📱 API Data Fetching App (Flutter)

This is a Flutter application that demonstrates how to fetch data from a REST API and display it in a clean and responsive user interface. The project focuses on API integration, JSON parsing, and handling asynchronous data efficiently.

🚀 Features
🌐 Fetch data from REST API
📊 Display data in list view
🔄 Real-time data loading
⏳ Loading indicator while fetching data
⚠️ Error handling (network/API issues)
🔁 Refresh functionality
🛠️ Technologies Used
Flutter (UI Toolkit)
Dart (Programming Language)
HTTP Package (API calls)
JSON Parsing
⚙️ How It Works

The app sends an HTTP request to a REST API, receives JSON response, converts it into Dart objects, and displays it in the UI.

📂 Project Structure
models/ → Data models
services/ → API logic
screens/ → UI screens
widgets/ → Reusable components
main.dart → App entry point
